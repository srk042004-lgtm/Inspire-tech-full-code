**PLACEHOLDER FOR MAIN VIDEO EMBED**

This practical class focuses on combining HTML table structure with advanced CSS styling to create a clear and well-organized school timetable. We will heavily utilize the **`colspan`** and **`rowspan`** attributes in HTML, and use CSS to ensure a professional, consistent look.

üß± Step 1: HTML Table Structure

A timetable requires complex cells that span multiple columns or rows.

  • **`colspan`**: Used to make a cell take up the space of multiple columns horizontally (e.g., spanning a lunch break across all days).
  • **`rowspan`**: Used to make a cell take up the space of multiple rows vertically (e.g., spanning a long lab session across two time slots).
Example Timetable HTML: 

<div class="timetable-container">
    <table class="timetable">
        <thead>
            <tr>
                <th>Time</th>
                <th>Monday</th>
                <th>Tuesday</th>
                <th>Wednesday</th>
                <th>Thursday</th>
                <th>Friday</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>9:00 - 10:00</td>
                <td>Math</td>
                <td>Physics</td>
                <td>Chemistry</td>
                <td>English</td>
                <td rowspan="2">Lab Session</td>
            </tr>
            <tr>
                <td>10:00 - 11:00</td>
                <td>History</td>
                <td>Urdu</td>
                <td>Islamiat</td>
                <td>Geography</td>
            </tr>
            <tr class="lunch-break">
                <td colspan="6">LUNCH BREAK (11:00 - 12:00)</td>
            </tr>
            <tr>
                <td>12:00 - 1:00</td>
                <td>CS</td>
                <td>Math</td>
                <td>CS</td>
                <td>Physics</td>
                <td>Math</td>
            </tr>
        </tbody>
    </table>
</div>

üé® Step 2: Essential CSS Styling

The key to fixing your "disturbed" table display lies in three crucial CSS properties:

**`border-collapse: collapse;`**
This **removes the double borders** that often make tables look messy, replacing them with a single, clean border.
**`width: 100%;`**
Ensures the table stretches to fill the available space of its container.
**`padding` on `th` and `td`**
Adds spacing inside the cells, which improves readability and professional appearance.
The Correct CSS to Fix Borders: 

.timetable {
  width: 100%;
  /* FIX: Merges double borders into single borders */
  border-collapse: collapse; 
  min-width: 600px; /* Ensures space for content */
}

.timetable th, .timetable td {
  /* FIX: Applies border to every cell edge */
  border: 1px solid #dee2e6;
  padding: 12px;
}

‚ú® Step 3: Advanced Styling (Zebra Striping & Highlighting)

To make the timetable easier to read, we use advanced selectors for styling:

  • **Zebra Striping**: Using the **`:nth-child(even)`** pseudo-class to apply a light background to every second row, dramatically improving legibility.
  • **Highlighting**: Creating specific classes (like `.lunch-break`) to highlight important rows using distinct background colors.
  • **Responsiveness**: Wrapping the entire table in a container with **`overflow-x: auto;`** allows the user to scroll horizontally on small phone screens, preventing the table from breaking the layout.
