CSS Introduction

What is CSS?

CSS stands for Cascading Style Sheets. It is a style-sheet language used to change the appearance, styling, layout, and visual presentation of your HTML pages.

Think of HTML as the structure of a house, and CSS as the paint, design, decoration, and layout of the rooms.

Why Do We Use CSS?

  • To style text, fonts, colors, spacing, and backgrounds
  • To create clean and attractive layouts
  • To make responsive websites for all devices
  • To avoid inline styling and keep code clean
  • To control styles of multiple pages from one file

How CSS Works?

CSS works by selecting HTML elements and applying styles to them. A CSS rule has:

  • Selector ‚Äî which element to style
  • Property ‚Äî what to change
  • Value ‚Äî the setting for the property
selector { property: value; }

Ways to Add CSS

1. Inline CSS
<h1 style="color: red;">Hello</h1>
2. Internal CSS
<style> h1 { color: blue; } </style>
3. External CSS (Best Practice)
<link rel="stylesheet" href="style.css">

What Can You Style With CSS?

  • Colors & backgrounds
  • Margins & padding
  • Fonts & text styles
  • Borders & shadows
  • Width & height
  • Flexbox & Grid layout
  • Animations & transitions
