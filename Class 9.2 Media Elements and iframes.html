Class 9.2: HTML Media Elements (Audio, Video) aur I-frames

Is class mein hum seekhenge ke apni website mein **multimedia (audio aur video)** kaise daalte hain. Hum **<audio>** aur **<video>** tags ka istemaal dekhenge, aur saath hi **<iframe>** tag se doosri websites ke content ko apne page mein kaise embed (daakhil) karte hain.

Video Lesson: Media Embedding in HTML

1. HTML Audio Aur Video Elements

HTML5 mein audio aur video files ko directly embed karna bohot aasan ho gaya hai. Ab humein kisi third-party plugin ki zaroorat nahi padti.

A. <video> Tag

Yeh tag aapki website par video content display karne ke liye istemal hota hai.

  • **`src`**: Video file ka path batata hai.
  • **`controls`**: Browser ke default video controls (play, pause, volume) ko show karta hai.
  • **`width`** aur **`height`**: Video player ki dimension (lambaai aur choudaai) set karta hai. 
                  
<!-- Local Video File Ko Daakhil Karna -->
<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Aapka browser video tag ko support nahi karta.
</video>

B. <audio> Tag

Yeh tag webpage par audio content (music, podcast, etc.) daalne ke liye istemal hota hai. Iska tareeqa <video> tag jaisa hi hai. 

                  
<!-- Local Audio File Ko Daakhil Karna -->
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
    Aapka browser audio tag ko support nahi karta.
</audio>

2. Iframes (<iframe> Tag)

An **iframe** (Inline Frame) ek HTML document ko dusre HTML document ke andar embed karne ke liye istemal hota hai. Aam taur par iska istemal **YouTube videos**, **Google Maps**, ya doosri websites ke chote hisson ko apni website par dikhaane ke liye kiya jata hai.

Iframe Ke Khaas (Important) Attributes

  • **`src`**: Woh URL (web address) jise aap embed karna chahte hain.
  • **`width`** aur **`height`**: Iframe ka size set karta hai.
  • **`frameborder`**: Purane browsers mein frame ki border dikhaane ke liye (modern CSS se control hoti hai). 
                  
<h3>Example: YouTube Video Embed Karna</h3>
<!-- YouTube khud hi iframe code deta hai -->
<iframe 
    width="560" 
    height="315" 
    src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
    title="YouTube video player" 
    frameborder="0" 
    allowfullscreen>
</iframe>
