In Class 7, you learned how to **select** HTML elements using methods like `querySelector()`. Now, we focus on the most powerful part of the DOM: **manipulation**, which means dynamically changing the content, attributes, and styles of those selected elements.

PART 1: Changing Content and HTML

These properties allow you to read or set the content of an element.

A. The Difference: `textContent` vs. `innerHTML`
1. `textContent` (Text Only)

Sets or returns the **text content** of the element and all its descendants. It treats any HTML tags as plain text, making it safer against security risks (XSS attacks).

2. `innerHTML` (HTML & Text)

Sets or returns the **HTML content** (including tags) of the element. Use this when you need to inject new HTML structure. 

// HTML: <div id="msg">Old Message</div>
const messageDiv = document.getElementById("msg");

// Injecting HTML using innerHTML
messageDiv.innerHTML = "<strong>New BOLD Message!</strong>";

// Setting only text content using textContent (safer)
messageDiv.textContent = "The link is here";
B. Changing Input Values

For form elements like `<input>` and `<textarea>`, you must use the **`.value`** property to get or set the data the user has typed in. 

// HTML: <input id="username" type="text" value="Default">
const inputField = document.getElementById("username");

// Get the current value
console.log(inputField.value); // Output: Default

// Set a new value
inputField.value = "Hassan_42";

PART 2: Changing Attributes and Styles

A. Managing Element Attributes

Attributes (like `src`, `href`, `class`, `id`) are managed using these methods:

  • **`.getAttribute(name)`**: Gets the current value of an attribute.
  • **`.setAttribute(name, value)`**: Sets or changes the value of an attribute.
  • **`.removeAttribute(name)`**: Removes an attribute completely. 
// HTML: <img id="logo" src="old.jpg">
const logo = document.getElementById("logo");

// Change the image source
logo.setAttribute("src", "new_logo.png");

// Add a title attribute
logo.setAttribute("title", "Inspire Tech Logo");
B. Controlling CSS Styles and Classes

The preferred way to change an element's appearance is by manipulating its CSS classes. For direct styling, use the `.style` property.

Direct Style Manipulation (`.style`)

Property names are written in **camelCase** in JavaScript (e.g., `background-color` becomes `backgroundColor`). 

// Set background color directly
messageDiv.style.backgroundColor = "yellow";
messageDiv.style.fontSize = "20px";
Manipulating Classes (`.classList`)

The best way to toggle visual state (e.g., making something hidden or red). 

// Assume 'error' class is defined in CSS
messageDiv.classList.add("error");    // Add the class
messageDiv.classList.remove("error"); // Remove the class
messageDiv.classList.toggle("hidden"); // Toggles on/off

PART 3: Creating and Removing Elements

To build dynamic UIs, you often need to create new elements and insert them into the DOM tree.

The Three Steps for Creation:
  1. **Create**: Use `document.createElement('tagName')`.
  2. **Configure**: Set its `textContent`, `id`, `class`, or attributes.
  3. **Attach**: Use an **append** method to place it inside an existing parent element. 
// 1. Create a new list item (<li>)
const newItem = document.createElement('li');

// 2. Configure it
newItem.textContent = "New Item Added";
newItem.classList.add("list-item");

// Find the parent (<ul> or <ol> element)
const parentList = document.querySelector("#my-list");

// 3. Attach it to the end of the parent
parentList.appendChild(newItem); // Inserts the element

**Removal:** Use the `remove()` method on the element you want to delete. 

// Remove the item we just created
newItem.remove();
