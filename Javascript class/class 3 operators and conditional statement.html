This class introduces the core tools programmers use to perform calculations and make decisions in code: **Operators** and **Conditional Statements**. Mastering these allows your programs to react dynamically to different data and inputs.

PART 1: JavaScript Operators

Operators are special symbols used to perform operations on operands (values and variables).

A. Arithmetic Operators (Calculations)

Used to perform basic mathematical calculations.

OperatorNameExampleResult
`+`Addition`x + y`Sum of x and y
`-`Subtraction`x - y`Difference of x and y
`*`Multiplication`x * y`Product of x and y
`/`Division`x / y`Quotient of x and y
`%`Modulus`10 % 3`1 (Remainder of division)
`**`Exponentiation`2 ** 3`8 ($2^3$)
`++`Increment`x++`Increases value by 1
`--`Decrement`x--`Decreases value by 1
 
let a = 5;
a++; // a is now 6
let result = a % 2; // result is 0

Used to assign values to variables. Compound assignments are shorthand for performing an operation and assignment at the same time.

C. Comparison Operators (The Decision Makers)

Used to compare two values. They always return a **Boolean** value (`true` or `false`).

OperatorNameResult
`==`Equal to (Loose equality)Compares value only (e.g., `5 == '5'` is true)
`===`**Strictly Equal**Compares **value AND type** (e.g., `5 === '5'` is false)
`!=`Not equal to (Loose)Compares value only

`>`Greater than

**IMPORTANT:** Always use **Strict Equality (`===` and `!==`)** in JavaScript. Loose equality (`==` and `!=`) often causes confusing bugs by automatically converting data types.

PART 2: Conditional Statements

Conditional statements allow your code to make decisions based on conditions (which are evaluated using comparison and logical operators).

A. The `if`, `else if`, and `else` Structure

The most common way to make decisions. The code inside the curly braces will run only if the condition is `true`.

let score = 85;

if (score >= 90) {
    console.log("Grade A");
} else if (score >= 80) {
    console.log("Grade B"); // This code runs
} else {
    console.log("Grade C");
}
B. The Ternary Operator (`? :`)

A shorthand way to write simple `if/else` statements. It is ideal for assigning a value based on a condition.

Syntax: `condition ? value_if_true : value_if_false;`

let age = 19;
let status = (age >= 18) ? "Adult" : "Minor"; 
// status is assigned the value "Adult"
C. The `switch` Statement

Used to perform different actions based on different conditions. It is cleaner than using many `else if` statements for a single variable.

  • **`case`**: Defines a possible value for the expression.
  • **`break`**: Crucial! It stops the execution inside the switch block. If omitted, the code will continue to run into the next `case`.
  • **`default`**: Code that executes if none of the cases match (like the `else` block).
let day = "Monday";
let activity;

switch (day) {
    case "Saturday":
    case "Sunday": // Multiple cases can share one block
        activity = "Weekend Relax";
        break;
    case "Monday":
        activity = "Start of Work"; // This code runs
        break;
    default:
        activity = "Midweek Routine";
}
console.log(activity); // Output: Start of Work
