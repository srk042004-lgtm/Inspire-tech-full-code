üì¶ Class 5: Arrays & Array Methods

Arrays are special variables that can hold more than one value at a time. They are the primary way to store ordered lists of data in JavaScript. Since arrays are a type of **Object** (non-primitive), they are powerful and come with many built-in functions called **Array Methods**.

PART 1: Array Basics (Declaration and Access)

A. Creating Arrays

The easiest and most common way to create an array is using **array literals** (square brackets `[]`). Arrays can hold different data types in the same list. 

// Array of strings (most common)
const students = ["Ali", "Sara", "Imran"];

// Array of mixed data types
const mixedArray = [10, "Text", true, null];

// Array property: .length
console.log(students.length); // Output: 3
B. Array Indexing and Access

Every item in an array has a numbered position called an **index**. Crucially, JavaScript (and most programming languages) uses **zero-based indexing**, meaning the first element is at index `0`. 

const fruits = ["Apple", "Banana", "Cherry"];
                            
// Accessing elements
console.log(fruits[0]); // Output: "Apple"
console.log(fruits[2]); // Output: "Cherry"

// Changing elements (Arrays are mutable)
fruits[1] = "Mango";
console.log(fruits); // Output: ["Apple", "Mango", "Cherry"]
**NOTE:** Although we typically declare arrays using `const`, we can still **change the contents** (elements) of the array, because `const` only prevents us from reassigning the *variable* `fruits` to a different array object.
C. Iterating over Arrays (Using Loops)

Loops (like the `for` loop you learned) are essential for processing every element in an array. 

const names = ["Ahmad", "Bilal", "Usman"];
                            
for (let i = 0; i < names.length; i++) {
    console.log(`Student ${i + 1}: ${names[i]}`);
}
// Output:
// Student 1: Ahmad
// ...

PART 2: Array Mutator Methods (Changing the Array)

Mutator methods change (mutate) the original array. These are used to add or remove elements from the ends of the array.

Method Description Return Value Example Result
`.push(item)` Adds one or more elements to the **end** of the array. New `length` of the array. Adds element.
`.pop()` Removes the **last** element from an array. The element that was removed. Removes last element.
`.unshift(item)` Adds one or more elements to the **beginning** of the array. New `length` of the array. Adds element at start.
`.shift()` Removes the **first** element from an array. The element that was removed. Removes first element.
Example: Adding and Removing 
let queue = ["A", "B"]; 

// Add to end
queue.push("C");     // queue is now ["A", "B", "C"]

// Remove from start
let next = queue.shift(); // next is "A", queue is now ["B", "C"]

// Add to start
queue.unshift("Z");  // queue is now ["Z", "B", "C"]

PART 3: Array Accessor Methods (Returning New Data)

Accessor methods do not modify the original array; instead, they return a new value or a new array based on the operation.

Method Description Example
`.join(separator)` Joins all array elements into a **string**. `['a', 'b'].join('-')` returns `"a-b"`
`.includes(value)` Checks if an array contains a specific value (returns `true` or `false`). `[1, 2, 3].includes(2)` returns `true`
`.indexOf(value)` Returns the **index** of the first occurrence of an element, or `-1` if not found. `['a', 'b'].indexOf('b')` returns `1`
`.slice(start, end)` Extracts a portion of an array into a **new array**. (Original array is unchanged). `[1, 2, 3, 4].slice(1, 3)` returns `[2, 3]`
**Tip:** The difference between `.push()` and `.slice()` is critical! `.push()` changes the original array, while `.slice()` leaves the original array alone and gives you a copy of a section.
