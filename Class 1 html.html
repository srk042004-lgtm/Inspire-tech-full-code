Introduction to HTML

HTML is the standard Markup language for creating Web Pages.

HTML stands for Hyper Text Markup Language.

HTML describes the structure of Web pages using Markup.

HTML Elements are the building blocks of HTML pages.

HTML Elements

What are HTML Elements?

HTML elements are the basic building blocks of a web page. An element usually consists of:

  • Opening tag (e.g. <p>)
  • Content (the text or other elements inside)
  • Closing tag (e.g. </p>)

Example of a simple HTML element:

<p>This is a paragraph element.</p>
Block-level Elements

Block-level elements always start on a new line and take up the full width.

  • <div> ‚Äî Division
  • <h1>...<h6> ‚Äî Headings
  • <p> ‚Äî Paragraph
  • <table> ‚Äî Table
Inline Elements

Inline elements do not start on a new line and take only the necessary width.

  • <span> ‚Äî Inline container
  • <a> ‚Äî Anchor (links)
  • <strong> ‚Äî Bold text
  • <em> ‚Äî Italic text

HTML Attributes

What are Attributes?

HTML attributes provide extra information about elements. They are always written inside the <start tag> and usually come in name="value" pairs.

  • Attributes are written in the opening tag only.
  • They usually define the behavior or appearance of an element.
  • Most attributes have default values.
Examples of Common Attributes
Attribute Description Example
href Specifies the URL for a link <a href="https://example.com">Visit</a>
src Specifies the source of an image or media file <img src="image.jpg" alt="My Image">
alt Provides alternate text for an image <img src="photo.jpg" alt="Beautiful scenery">
id Specifies a unique identifier for an element <div id="main">Content</div>
class Specifies one or more class names (used with CSS/JS) <p class="highlight">Text</p>
style Applies inline CSS styles to an element <p style="color:blue;">Hello</p>
title Specifies extra information shown as a tooltip <abbr title="World Health Organization">WHO</abbr>

Steps to Create an HTML Document

Start by opening a plain Text Editor on your computer. You can use any text editor available on your operating system, such as Notepad, Notepad++, or Visual Studio.

In the text editor, create a new file and begin by adding the HTML document structure.

Within the HTML structure, include the <html> tags to enclose your entire HTML content. Inside the <html> tags you will have the <head> and <body> sections.

In the <head> section, you can include metadata about the document such as: - Title - Character encoding - Linked CSS Stylesheets - JavaScript files

In the <body> section, add the visible content of your webpage. You can use various HTML tags to structure and format your content, such as: - Headings - Paragraphs - Lists - Images - Links

Save the file with an appropriate name and the .html extension. For example: index.html or mypage.html.
Note: Attributes make HTML more powerful by adding extra meaning or control to elements.
