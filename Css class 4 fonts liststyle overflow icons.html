üìù Class 4: Fonts, List Styles, Overflow & Icons

In this class, we dive into advanced text presentation, list formatting, managing content boundaries, and adding visual appeal with icons. These elements are key to creating a polished and user-friendly design.

üî§ CSS Font Styling Properties

The **`font-`** properties allow you to control every aspect of your text typography.

Key Font Properties:
  • **`font-family`**: Specifies the font face (e.g., `'Arial', sans-serif`). Always use **fallback fonts** (generic families like `serif`, `sans-serif`) in case the primary font isn't available.
  • **`font-size`**: Sets the size of the text. Recommended units include **`px`** (absolute), **`em`** (relative to parent), or **`rem`** (relative to root HTML element).
  • **`font-weight`**: Sets the thickness or boldness (e.g., `normal`, `bold`, or numeric values from `100` to `900`).
  • **`font-style`**: Sets the style of the font (e.g., `normal`, **`italic`**, or `oblique`).

p {
  font-family: 'Helvetica Neue', Arial, sans-serif;
  font-size: 1.1rem;
  font-weight: 400;
  font-style: italic;
}

üìã List Styling (`list-style`)

CSS can transform basic bulleted and numbered lists (`

    ` and `
      `) into custom-styled navigation or clean content blocks. The shorthand property is **`list-style`**.

      Main Properties:
      Property Purpose / Key Values
      **`list-style-type`** Sets the marker type: `disc`, `circle`, `square`, `decimal`, `lower-alpha`, `upper-roman`, or **`none`** (to remove markers).
      **`list-style-position`** Sets marker position: **`outside`** (default, hanging indent) or **`inside`** (part of the list item text flow).
      **`list-style-image`** Uses an image as the marker: `url('path/to/image.png')`. 
      
ul {
  list-style: square inside; /* Shorthand: type and position */
  padding-left: 0; /* Remove default padding for outside position */
}

ol {
  list-style-type: lower-roman; /* i, ii, iii... */
}

üì¶ The `overflow` Property (Managing Content)

The **`overflow`** property controls what happens when content is too large to fit within an element's defined height and/or width.

Key Overflow Values:
  • **`visible`** (Default): Content is **not** clipped and flows outside the element box.
  • **`hidden`**: Overflowing content is **clipped** and becomes invisible. No scrollbar is shown.
  • **`scroll`**: Always adds a scrollbar (both horizontal and vertical), even if the content doesn't overflow.
  • **`auto`**: Adds a scrollbar **only when necessary** for the content to be viewed. This is often the best choice.
This box has a fixed width and height (300px x 100px). When this text content is long, the `overflow: auto;` property automatically adds a scrollbar so the user can read the rest of the text without breaking the layout. This is very important for fixed-size containers like sidebars or content panels. 

.scrollable-content {
  height: 150px;
  overflow-y: auto; /* Scroll vertically only when needed */
  overflow-x: hidden; /* Prevent horizontal scroll */
}

‚≠ê Adding & Styling Icons

Icons (like Font Awesome, Bootstrap Icons, or Google Material Icons) are often implemented using specialized web fonts or SVG graphics. They are treated like text, which makes them very easy to style with CSS.

Styling Icons:

After linking an icon library (like Font Awesome in this page's ``), you can style the icon elements (``) using the same properties as text:

  • **`color`**: Changes the icon's color.
  • **`font-size`**: Changes the icon's size.
  • **`margin`/`padding`**: Adjusts spacing around the icon.

**Example:** In the code, we style the Font Awesome icon for the gear: 


.icon-gear {
  color: #ffc107; /* Gold color */
  font-size: 1.5rem;
  margin-left: 5px;
}
