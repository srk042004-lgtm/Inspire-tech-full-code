**CSS (Cascading Style Sheets)** is the language used to **style** a web page, defining how HTML elements should be displayed. It controls colors, fonts, layout, and visual effects, turning basic HTML structure into a beautiful, engaging website.

‚ú® CSS Syntax: The Rule-Set

A CSS rule-set consists of a **selector** and one or more **declarations**. 


/* Selector points to the HTML element you want to style */
p {
  /* Declaration is property: value; */
  color: blue; 
  font-size: 16px; 
}
üéØ Basic Selectors

Selectors are the mechanisms used to find and select the HTML elements you want to style.

Selector Type Syntax Description
**Element** `p { ... }` Selects all HTML elements of that type (e.g., all paragraphs).
**ID** `#header { ... }` Selects the element with a specific ID. **Use sparingly; must be unique.**
**Class** `.button { ... }` Selects all elements with a specific class. **Best for repeated styling.**
