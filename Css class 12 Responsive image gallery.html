Image galleries are essential for portfolios and commercial sites. Creating a truly **responsive gallery** that fluidly changes the number of columns based on screen size is best achieved using **CSS Grid**. This class introduces the basics of Grid and combines it with Media Queries for a perfect gallery layout.

üåê Step 1: CSS Grid Fundamentals

**CSS Grid** is a two-dimensional layout system that allows you to define rows and columns, making it ideal for overall page layouts and complex components like galleries.

Key Grid Properties:
  • **`display: grid;`**: Applied to the parent container to activate the Grid layout.
  • **`grid-template-columns`**: Defines the number and size of the columns. Example: `grid-template-columns: 1fr 1fr 1fr;` creates three equal columns.
  • **`gap`**: Sets the spacing between the grid items (both rows and columns).
**Grid vs. Flexbox**: Flexbox is for laying out items in a single dimension (row OR column), while Grid is for two dimensions (rows AND columns) simultaneously.

üìê Step 2: Auto-Placing Items with `minmax` and `auto-fit`

To make a gallery automatically adjust the number of columns, we use the following powerful combination within `grid-template-columns`:

**`repeat()`**
A function to repeat a column definition multiple times.
**`auto-fit`**
Tells the browser to automatically calculate how many columns will fit.
**`minmax()`**
A function that defines a minimum and maximum size for a column. Example: `minmax(200px, 1fr)` ensures the column is at least 200px wide, but can grow to fill space (`1fr`).
The Responsive Gallery Magic Code:

.gallery-grid {
  display: grid;
  /* Columns will adjust based on available space: 
     - They will try to fit as many as possible (auto-fit).
     - Each column will be at least 250px wide.
     - They will take up equal remaining space (1fr). */
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

üñºÔ∏è Step 3: Image Sizing and Aspect Ratios

In a grid gallery, it's common to maintain a consistent look regardless of the image's original dimensions.

Crucial Image Styling for Grids:
  • **Full-Cell Fill**: Set the image to **`width: 100%;`** and **`height: 100%;`** inside its grid item container.
  • **`object-fit: cover;`**: This property forces the image to cover the entire container area, cropping any excess part so the image maintains its aspect ratio without stretching. This is superior to simply setting a fixed height.
  • **Image Hover Effects**: Add a slight **`transform: scale(1.05);`** effect on hover, coupled with a `transition`, to give the gallery an interactive, professional feel.
