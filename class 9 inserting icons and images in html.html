Inserting Images and Icons in HTML

Adding images and icons is an important part of web design. Images help make your website more attractive, while icons improve user experience by representing actions or ideas visually.

You can use the <img> tag to insert an image into a webpage. This tag is self-closing and requires the src and alt attributes.

Example:
<img src="images/photo.jpg" alt="Beautiful landscape" width="400" height="250">
  • src: The path of the image (local or URL).
  • alt: Alternate text displayed if the image doesn‚Äôt load.
  • width/height: Set image size in pixels or percentages.

‚≠ê Inserting Icons

Icons are used to represent ideas and actions visually. You can add icons using services like Font Awesome or Bootstrap Icons.

Using Bootstrap Icons

Include this link in your <head> section:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.5/font/bootstrap-icons.css">

Then use icons like this:

<i class="bi bi-alarm"></i> Alarm Icon
Using Font Awesome

Add this line in your <head>:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Then use:

<i class="fa-solid fa-camera"></i> Camera Icon

You can customize icon color and size using CSS or inline styles:

<i class="fa-solid fa-heart" style="color:red; font-size:30px;"></i>
