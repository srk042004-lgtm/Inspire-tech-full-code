HTML Classes, IDs and Bookmarks

In HTML, Classes and IDs are used to identify and style elements easily. They help web developers organize and control how specific parts of a webpage look and behave.

üîπ HTML Class Attribute

The class attribute is used to assign one or more class names to an element. It allows CSS or JavaScript to select and style multiple elements with the same class.

Example:
<p class="highlight">This is a highlighted paragraph.</p>

üîπ HTML ID Attribute

The id attribute is used to uniquely identify a single HTML element on the page. An ID must be unique ‚Äî no two elements should have the same ID.

Example:
<p id="intro">This is a unique paragraph.</p>

üîπ Using Classes and IDs in CSS

CSS uses . (dot) for classes and # (hash) for IDs.

Example CSS:
.highlight { color: blue; font-weight: bold; }
#intro { color: red; font-size: 20px; }

üîπ HTML Bookmarks

Bookmarks let users jump to specific sections within the same webpage using the id attribute.

Example:
<a href="#chapter2">Go to Chapter 2</a>
<h2 id="chapter2">Chapter 2</h2>

Bookmarks are very useful for creating table of contents, navigation menus, and internal linking.
