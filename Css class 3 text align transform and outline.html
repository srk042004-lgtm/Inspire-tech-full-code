üé• Class 5: Outline, Text Alignment, and Transformation

This class focuses on styling text and controlling the visual border that is applied outside of the Box Model: the **Outline**. Mastering these properties allows you to control typography and enhance accessibility.

üè∑Ô∏è CSS Outline Property

The **Outline** is a line drawn **outside** the element's border, and it does **not** take up space or affect the element's position. It is mainly used for **accessibility** to indicate that an element is in focus (e.g., when tabbing through form fields).

Key Outline Properties:
Property Description
`outline-style` Sets the style of the outline (e.g., `solid`, `dashed`, `dotted`).
`outline-width` Sets the thickness of the outline (e.g., `5px`).
`outline-color` Sets the color of the outline.
**Shorthand** `outline: 3px solid green;` (width style color)
‚ö†Ô∏è **Accessibility Warning:** While `outline: none;` can hide the default focus ring, **avoid removing outlines** completely, as it makes your site unusable for keyboard users. Instead, style the outline to match your design.

‚ÜîÔ∏è Text Alignment (`text-align`)

The `text-align` property controls the **horizontal alignment** of inline content (like text and images) inside a block-level element (like a `<p>` or `<div>`).

Available Values:
  • **`left`**: Aligns content to the left (default for most languages).
  • **`right`**: Aligns content to the right.
  • **`center`**: Centers the content horizontally within the element.
  • **`justify`**: Stretches the lines of text so that both the left and right edges are aligned (like in newspapers).

.header {
  text-align: center; /* Centers the text inside the header */
}

üÖ∞Ô∏è Text Transformation (`text-transform`)

The `text-transform` property is used to convert text to different capitalizations without needing to change the HTML code itself. This is useful for consistent styling.

Available Values:
Value Example Output
**`uppercase`** THIS IS ALL CAPS.
**`lowercase`** this is all lowercase.
**`capitalize`** Capitalizes The First Letter Of Each Word.
**`none`** The text remains unchanged (default). 

.button {
  text-transform: uppercase; 
}
üí° **Tip: Font Properties:** Other crucial text properties to explore include `font-size`, `font-weight`, `line-height`, and `text-decoration` (for removing underlines from links).
