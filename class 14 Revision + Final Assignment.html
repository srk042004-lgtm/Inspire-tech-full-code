Class 14: HTML Comprehensive Revision and Practice

Mubarak ho! Aapne HTML ka aadha safar pura kar liya hai. Yeh class pichle sabhi topics (Class 1 se 13 tak) ko mazboot karne ke liye hai. Hum **core concepts** ko revise karenge aur complex web structure banane ki practice karenge.

Video Lesson: Quick HTML Review and Best Practices

1. Deep Dive Revision: Tags Aur Attributes

A. Core Structure & Metadata (Class 1, 3)

Har page <!DOCTYPE html> se shuru hota hai. **Head** section mein metadata (`<meta>`) aur **Title** zaroori hain. **Body** tag mein poora visible content aata hai. 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Practice Page</title>
</head>
<body>
    <!-- Content Here -->
</body>
</html>

B. Text Formatting and Quotations (Class 4, 6)

Headings **H1** se **H6** tak hote hain (sirf ek H1 use karna best practice hai). Use **`<strong>`** for importance, **`<em>`** for emphasis. Quotations ke liye **`<blockquote>`** (long quote) aur **`<q>`** (inline quote) ka use karen. 


<h1>Main Title of the Document</h1>
<p>This text is normal, but <strong>this part is important</strong>.</p>
<blockquote cite="Source URL">
    A long quotation goes here.
</blockquote>

C. Links, Classes, aur IDs (Class 8)

Links (`<a>`) mein **`href`** destination batata hai. **`target="_blank"`** se naye tab mein link khulta hai. **Classes** (`class="btn"`) styling ke liye, aur **IDs** (`id="section1"`) unique element identify karne ya **bookmarks** banane ke liye use hote hain. 


<a href="https://example.com" target="_blank">Visit Site</a>
<a href="#contact-form">Jump to Contact</a> <!-- Bookmark -->
<div id="contact-form" class="main-container">...</div>

D. Images and Icons (Class 9)

Image tag **`<img src="" alt="">`** self-closing hota hai. **`alt`** attribute screen readers aur jab image load na ho to zaroori hota hai. Icons ke liye Font Awesome ya SVG ka istemaal kar sakte hain. 


<img src="my-photo.jpg" alt="A professional portrait" width="300">
<i class="fas fa-star"></i> <!-- Font Awesome Icon -->

E. Tables (Class 10)

Tables mein **`<thead>`**, **`<tbody>`**, aur **`<tfoot>`** structure dete hain. **`colspan`** aur **`rowspan`** se cells ko merge kiya jata hai. 


<table border="1">
    <thead><tr><th colspan="2">Product Details</th></tr></thead>
    <tbody>
        <tr><td>Price</td><td>$50</td></tr>
    </tbody>
</table>

F. Forms and Input Controls (Class 11/12)

Forms mein **`<label>`** hamesha **`<input>`** se `for` aur `id` attributes ke zariye link hona chahiye. Different **`type`** attributes (text, email, radio, password) use hote hain. **`required`** attribute se validation lagai jati hai. 


<form>
    <label for="user_email">Email:</label>
    <input type="email" id="user_email" required placeholder="Enter valid email">
    <input type="submit" value="Sign Up">
</form>

2. Practice Challenge 1: E-commerce Product Card

Aapko ek single **E-commerce Product Detail Card** banana hai jo neeche diye gaye sabhi elements ko combine karega:

Task: Product Detail Page Skeleton

  1. **Heading:** Product ka naam (`<h1>`) aur brief description (`<p>`).
  2. **Image & Caption:** Product ki ek image (`<img>`) use karen. Image ke neeche image ka naam **bold** karke likhen.
  3. **Features Table:** Product features ke liye ek table banayein (e.g., Colour, Warranty, Delivery Time). Delivery Time ke liye `colspan` use karen.
  4. **Order Form:** Ek chota form jismein sirf **Quantity** (`type="number"`) aur **Add to Cart** (`type="submit"`) button ho.
  5. **Internal Link:** Table ke upar ek internal bookmark link dein jo form tak jump kare.

**Goal:** Code mein class attributes ka istemaal zaroor karen taki woh aage styling ke liye ready ho.

3. Practice Challenge 2: Detailed Feedback Form

Aapko ek detailed feedback form banana hai. Yeh form forms, links, aur formatting ko acchi tarah use karega.

Task: Advanced Feedback Form

  1. **Personal Info:** Full Name (`text`), Email (`email` - required), Password (`password`), Aur Phone Number (`tel` - `placeholder` use karen).
  2. **Rating:** Ek question banayein: "How was your experience?" Aur teen options dein: Excellent, Average, Poor (use **`type="radio"`** aur har input ka `name` same hona chahiye).
  3. **Checkboxes:** Ek question banayein: "Which topics should we cover next?" Aur teen options dein: HTML Forms, CSS Basics, JS Basics (use **`type="checkbox"`**).
  4. **External Link:** Form ke neeche ek link dein jo "Privacy Policy" page par jaye (`target="_blank"`).
  5. **Submit Button:** Ek submit button dein.

**Goal:** Har input ko uske corresponding **`<label>`** tag ke saath zaroor jodein (using `for` and `id`).

4. Common Mistakes Aur Unka Hal

Mistake 1: Missing Closing Tags

**Galti:** `<p>This is a paragraph.` (Closing `</p>` bhool jana). **Sahi Hal:** Hamesha closing tag use karen. **VS Code** auto-closing tags mein madad karta hai.

Mistake 2: Missing `alt` Attribute in Images

**Galti:** `<img src="logo.png">` **Sahi Hal:** `alt` attribute bahut zaroori hai. `<img src="logo.png" alt="Company Logo">`

Mistake 3: Incorrectly Linking Label and Input

**Galti:** `<label>Name</label> <input type="text" id="name">` (Label mein `for` missing hai). **Sahi Hal:** `<label for="full_name">Name</label> <input type="text" id="full_name">` (`for` aur `id` match hone chahiye).
